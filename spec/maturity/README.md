# Maturity (pre-seed → seed → Series A)

Use this folder when **stage matters more than category**. Each stage is an **ordered list** of milestones; every row links to the same **`checklist.md`** under `spec/goals/`.

| Stage | Human list | What it proves (short) |
|-------|------------|-------------------------|
| **Pre-seed** | [pre-seed.md](pre-seed.md) | Credible problem, team, first build and learning |
| **Seed** | [seed.md](seed.md) | Traction or strong de-risking; seed round and early GTM |
| **Series A** | [series-a.md](series-a.md) | Repeatable motion; scale what works |

**For tools:** [`index.yaml`](index.yaml) lists `milestone_ids` per stage.

**Note:** Stages overlap in real life; this is a **default ordering**, not a law. Rare or edge milestones (e.g. wind-down) stay in the main catalog only.
